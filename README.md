# C# and .NET Learning Repository

## 📘 Introduction
This repository contains notes, examples, and small projects developed while learning **C#** and the **.NET** framework.  
The goal is to understand the fundamentals of programming with C# and how to build applications using .NET.

---

## 🎯 Objectives
- Learn the basics of C# programming
- Understand object-oriented programming (OOP)
- Explore the .NET ecosystem
- Build small practical examples

---

## 🧩 Topics Covered

### 🔹 C# Basics
- Variables and data types
- Operators
- Conditional statements (if, switch)
- Loops (for, while, foreach)

### 🔹 Object-Oriented Programming
- Classes and objects
- Encapsulation
- Inheritance
- Polymorphism

### 🔹 Methods and Functions
- Method declaration
- Parameters and return types
- Method overloading

### 🔹 Collections
- Arrays
- Lists
- Dictionaries

### 🔹 Exception Handling
- Try / Catch / Finally
- Custom exceptions

### 🔹 LINQ (Language Integrated Query)
- Filtering data
- Sorting
- Selecting data

### 🔹 .NET Fundamentals
- What is .NET
- Project structure
- .NET CLI basics

### 🔹 File Handling
- Reading and writing files
- Working with streams
